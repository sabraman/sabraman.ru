## Details

- Output: `components/ios-alert-dialog.tsx` so the final import stays `@/components/ios-alert-dialog`.
- Dependencies: Installs both the alert component and the supporting `components/ui/dialog.tsx` primitive in one registry item.
- Behavior: Ships with the original iOS 6 button gradients and gloss treatment, while still exposing a composable React API for real confirmation flows.
- Variants: The docs example covers the three core layouts: message-only, horizontal actions, and stacked actions.
