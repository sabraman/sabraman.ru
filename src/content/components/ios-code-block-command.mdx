## Installation

Install the direct item URL or use the Sabraman alias. The component is self-contained, so consumers do not need extra helper hooks or tab primitives.

## Usage

Pass one or more package-manager commands directly, or use `convertNpmCommand` to generate the full bun, npm, pnpm, and yarn set from a single npm-style command.

## Details

- Output: `components/ios-code-block-command.tsx` so the final import stays `@/components/ios-code-block-command`.
- Behavior: Persists the selected package manager in local storage, copies the active command to the clipboard, and exports a converter helper so one npm-style command can drive the full tab set.
