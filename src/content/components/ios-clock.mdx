## Details

- Output: `components/ios-clock.tsx` so the final import stays `@/components/ios-clock`.
- Behavior: Defaults to live local time, but accepts a `date` prop for fixed snapshots and previews.
- Variants: Ships with both `day` and `night` treatments from the Figma clock icon set.
- Integration: The same component now powers the live Clock icon on the portfolio iPhone home screen.
