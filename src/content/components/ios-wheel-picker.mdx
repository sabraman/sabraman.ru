## Usage

The registry block exports only the shell container and the column primitive. That keeps it flexible for time pickers, counters, unit selectors, or any legacy slot-machine UI built on the same wheel foundation.

## Details

- Output: `components/ios-wheel-picker.tsx` so the final import stays `@/components/ios-wheel-picker`.
- Dependency: Built on top of `@ncdai/react-wheel-picker` while keeping the shell and wheel geometry under your control.

## Notes

- The shell is intentionally opinionated, but the wheel columns stay composable.
- The row ratios and inset gloss are tuned to the original iOS picker chrome instead of a generic wheel control.
- Real example files power the docs so the copied markdown and the on-page code stay synced.
