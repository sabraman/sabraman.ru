## Details

- Output: `components/ios-bar-button.tsx` so the final import stays `@/components/ios-bar-button`.
- Layouts: Supports `text`, `icon`, `text-icon`, and `backward` for the original navigation patterns.
- Variants: Ships with the default silver-blue, accent blue, and destructive red gradients from the iOS 6 kit.
